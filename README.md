# PiXL Learning: Technical help page

A page used for my PiXL learning

## Keyboard Shortcuts

These are my keyboard shortcuts

## How to print
```
ctrl + p
```
## Copying Text in a Document

If you want to copy text:

```
ctrl + c
```
## Pasting text
```
ctrl + v
```

## Making a new window
```
ctrl + n
```
## Making a new tab
```
ctrl + t
```
## New folder
```
ctrl + shift + n
```
